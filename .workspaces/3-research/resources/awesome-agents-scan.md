# Awesome Agents Scan

Questa scan raccoglie liste curate e repository ad alta densità di progetti utili alla whiteboard.

## 1) Shubhamsaboo/awesome-llm-apps
- URL: https://github.com/Shubhamsaboo/awesome-llm-apps
- Descrizione: raccolta enorme di app LLM e AI agents con esempi pratici.
- Sezioni utili:
  - AI Agents
  - Starter AI Agents
  - Advanced AI Agents
  - Autonomous Game Playing Agents
  - Multi-agent Teams
  - Voice AI Agents
- Uso per i beads: ottimo per bead “starter/advanced/voice/multi-agent”.

## 2) hesreallyhim/awesome-claude-code
- URL: https://github.com/hesreallyhim/awesome-claude-code
- Descrizione: lista curata dell’ecosistema Claude Code: skills, hooks, slash commands, orchestrators, app e plugin.
- Sezioni utili:
  - Agent Skills
  - Workflows & Knowledge Guides
  - Tooling
  - IDE Integrations
  - Usage Monitors
- Uso per i beads: bead ad alta priorità per skills, workflows e tooling del coding agent.

## 3) VoltAgent/awesome-agent-skills

- Uso per i beads: bead “skills library” e bead per vendor/team skill packs.
